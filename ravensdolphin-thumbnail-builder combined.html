save/export:
  1. Click save PNG to download the thumbnail at exactly 1280×720
  2. Use save JPG or save WebP if you want smaller file sizes
  3. If your browser blocks the download, allow downloads for this local file and click again

manual screenshot fallback at exactly 1280 × 720:
  1. Open this file in Chrome or Firefox
  2. Press F12 to open DevTools → click the Toggle Device Toolbar icon (or Ctrl+Shift+M)
  3. Set dimensions to 1280 × 720 and zoom to 100%
  4. Right-click the preview area → Capture screenshot (Chrome) or use a screen capture tool
  5. Alternatively: set your browser zoom to 100%, resize the window so the preview fills it, and use your OS screenshot tool cropped to the thumbnail box

editing: drag the text, badge, character, or logo directly on the preview to place them where you want.

text, colors, sizes, layout mode, and dragged positions are remembered on this browser. Uploaded images stay temporary so local storage does not get overloaded.


tips: character PNGs with transparent backgrounds work best (export your green screen footage as PNG sequence in OBS or CapCut). Game logos are usually available as transparent PNGs from the official press kits.
