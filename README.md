# # ravensdolphin Thumbnail Builder

A lightweight local thumbnail builder for gaming, YouTube, and social content.

This project runs as a single HTML file in the browser and is designed for fast custom thumbnail creation without sign-ins, subscriptions, or bloated editing tools.

## Features

- Live thumbnail preview
- `16:9` and `9:16` layout modes
- PNG, JPG, and WebP export
- Draggable text, badge, character, and logo
- Layer ordering controls
- Separate eyebrow and title text controls
- Font selection with a small practical list
- Independent eyebrow and title colors
- Accent color and overlay controls
- Clean mode toggle
- Saved settings in the browser with `localStorage`

## Why This Exists

A lot of thumbnail tools are either too limited, too template-heavy, or too bloated for quick custom work. This builder is meant to stay simple: open the file, drop in your assets, move things where you want them, and export.

## How To Use

1. Open `ravensdolphin-thumbnail-builder combined.html` in your browser.
2. Upload a background, character cutout, and logo if you want them.
3. Edit the eyebrow and main title text.
4. Drag elements into place.
5. Adjust colors, fonts, sizing, and layer order.
6. Export as PNG, JPG, or WebP.

## Notes

- Settings are saved in your browser.
- Uploaded images are temporary for the current session.
- `Bebas Neue` is included as an optional style because it matches the original look.
- Most font choices are intentionally kept simple and practical for thumbnail work.

## Status

This is a focused single-purpose tool, not a full design platform. The goal is speed, simplicity, and useful control for thumbnail creation.
